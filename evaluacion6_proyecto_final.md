## Curso: Desarrollo y Diseño Web + IA

## Ponderación: 20%
## Tipo: Evaluación docente (proyecto + presentación + defensa)

## DESCRIPCIÓN DEL PROYECTO FINAL

Desarrolle una aplicación web completa que integre inteligencia artificial de manera significativa y demuestre todas las habilidades adquiridas durante el curso. El proyecto debe ser original, funcional y mostrar un nivel profesional de desarrollo.

### Requisitos técnicos obligatorios:
1. **Aplicación web completa** con frontend (HTML/CSS/JS) y backend (PHP)
2. **Integración real de IA** (API externa, modelo preentrenado, o algoritmo básico)
3. **Base de datos MySQL** con estructura normalizada
4. **Autenticación de usuarios** (registro, login, sesiones PHP)
5. **Interfaz responsive y accesible**
6. **Despliegue funcional** (local con XAMPP/WAMP o hosting)
7. **Código en GitHub** con historial de commits significativo
8. **Documentación completa** (README, setup instructions, manual técnico)

### Áreas de aplicación sugeridas (ejemplos):
- **Asistente educativo con IA** para feedback personalizado (PHP + API IA)
- **Generador de contenido automático** (texto con IA, almacenado en MySQL)
- **Sistema de recomendación inteligente** (algoritmo básico + PHP/MySQL)
- **Herramienta de análisis de datos** con visualizaciones y procesamiento IA
- **Chatbot especializado** integrado en sitio web PHP
- **Sistema de evaluación automática** con IA para retroalimentación
