## DESCRIPCIÓN DE LA EVALUACIÓN

Cree un mapa conceptual digital que integre y relacione los conceptos clave del desarrollo web completo, demostrando comprensión de frontend, backend, bases de datos y su interacción en aplicaciones web modernas.

### Requisitos del mapa conceptual:
1. **Formato digital** (Miro, Lucidchart, Draw.io, FigJam, o similar)
2. **Estructura jerárquica** con concepto central
3. **Mínimo 30 nodos/conceptos** relevantes
4. **Conexiones etiquetadas** que expliquen relaciones
5. **Código de colores** para categorías diferentes
6. **Ejemplos prácticos** en nodos relevantes
7. **Leyenda** que explique símbolos y colores
8. **Exportación** en PDF y formato editable original

### Conceptos que debe incluir (mínimo):
#### FRONTEND:
- HTML5, CSS3, JavaScript ES6+
- Responsive design, accesibilidad
- DOM manipulation, eventos
- APIs REST consumo (Fetch, Axios)

#### BACKEND (PHP):
- PHP sintaxis, variables, estructuras control
- Conexión MySQL (MySQLi, PDO)
- Formularios, validaciones, sesiones
- CRUD completo, operaciones básicas
- API REST con PHP, endpoints

#### BASES DE DATOS:
- MySQL estructura, tablas, relaciones
- SQL básico (SELECT, INSERT, UPDATE, DELETE)
- Normalización, claves primarias/foráneas
- Consultas con JOINs, subconsultas

#### SEGURIDAD:
- SQL injection prevention (prepared statements)
- XSS protection (htmlspecialchars)
- Validación cliente/servidor
- Autenticación, autorización básica

#### ARQUITECTURA:
- Cliente-servidor, peticiones HTTP
- Separación lógica/presentación
- Patrones MVC (concepto básico)
- Despliegue (XAMPP, hosting)

### Proceso de evaluación:
1. **Revisión inicial:** Verificar completitud de requisitos
2. **Análisis profundidad:** Evaluar comprensión conceptual, no solo listado
3. **Verificar relaciones:** Que conexiones sean significativas, no aleatorias
4. **Evaluar explicación:** Coherencia entre mapa y explicación escrita

### Criterios clave:
- **Comprensión vs. memorización:** ¿Demuestra entender relaciones o solo repite?
- **Pensamiento sistémico:** ¿Ve el ecosistema como conjunto interconectado?
- **Aplicabilidad:** ¿Puede conectar conceptos con práctica real?

### Para evaluación diferenciada:
- **Enfoque en fortalezas:** Algunos estudiantes pueden destacar en pensamiento visual
- **Alternativas para dificultades:** Ofrecer formatos alternativos según necesidades
